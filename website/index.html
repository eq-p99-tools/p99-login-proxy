Setup Instructions

1

Download & Install

Download the latest version from the link above and extract it to your preferred location. It will attempt to locate your EverQuest directory automatically, but if it is non-standard then you may need to put the application executable inside the EverQuest directory.

2

Configuration

Run the application and paste your API Token (in Discord, run "/sso get"). You may need to go to the Advanced tab and click Refresh Cache the first time.

3

Start Playing

Click the "Launch EQ" button (or start the game however you normally do) and simply log in with guild bot's name or role with any password, even just a single letter. Passwords don't matter anymore! For example:
